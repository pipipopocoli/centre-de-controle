PHASE 1 Fondations & Identité

~30 Heures

L'objectif est de briser les hardcodages et de donner une "âme" aux agents.

  • Agent Registry : Remplacer le if/else géant par un agents.json dynamique.
  • Hiérarchie L0/L1/L2 : Définir qui est boss (Clems), qui est lead (Victor/Léo), qui est worker.
  • SOUL.md : Créer les fichiers de personnalité pour chaque agent.
  • Skill Engine : Charger les compétences (Python, React, etc.) dynamiquement dans le prompt.

PHASE 2 Intelligence & Connexions

~35 Heures

Rendre le système robuste aux pannes et capable de se souvenir des choses simples.

  • Platform Router : Un système capable de switcher entre Codex, Antigravity et Ollama si une API tombe.
  • Memory (FTS5) : Une mémoire basée sur les mots-clés (rapide, léger, sans serveur).
  • Org Chart UI : Visualiser l'équipe sous forme de graphe interactif.

PHASE 3 Puissance & Autonomie

~40 Heures

Le passage au "Heavy Metal" : IA locale et mémoire sémantique profonde.

  • Vector Store (ChromaDB) : Recherche par "sens" et plus seulement par mots-clés.
  • Local LLM (Ollama) : Faire tourner des agents gratuits et privés sur ton Mac.
  • Cost Tracking : Savoir exactement combien coûte chaque tâche.
