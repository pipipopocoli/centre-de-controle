## Recommended Wording (Cohesive Set)

Use this set for a punchy, agile feel:
- **Timeline** -> **Cap**
- **Now** -> **Focus**
- **Next** -> **Suite**
- **Risks** -> **Alertes**
- **Objective** -> **Cible**

*Why?* Monosyllabic or very short words allow for quick scanning and fit well in tight UI spaces. "Focus/Suite/Alertes" creates a clear action-oriented rhythm.

## QA Checklist (Simple 5-Point)

1.  **Lisibilité & Contraste**
    - [ ] Le texte est-il lisible sur fond sombre/clair?
    - [ ] La hiérarchie visuelle (titre vs contenu) est-elle claire?

2.  **Espace & Alignement**
    - [ ] Le texte respire-t-il? (Marges suffisantes)
    - [ ] Les éléments sont-ils bien alignés (grille)?

3.  **Cas Limites (Overflow)**
    - [ ] Que se passe-t-il si le texte est très long? (Ellipsis `...` ou retour à la ligne?)
    - [ ] Que se passe-t-il si la liste est vide? (Message "Rien à signaler" ou vide?)

4.  **Consistance**
    - [ ] La terminologie est-elle identique partout? (Pas de mélange Anglais/Français)
    - [ ] Les majuscules/minuscules sont-elles harmonisées?

5.  **Feedback & État**
    - [ ] L'interface indique-t-elle si les données sont à jour?
    - [ ] Y a-t-il un indicateur de chargement si nécessaire?
